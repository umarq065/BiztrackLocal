# **App Name**: BizTrack Pro

## Core Features:

- Month/Year Filter: Select month and year from dropdowns to filter data. Year range: 2021-2030.
- Manage Income Sources: Add, edit, and delete income sources with associated dates.
- Manage Clients: Add, edit, and delete clients with details like username, name, email, and source.  View clients by income source.
- Manage Orders: Add orders with date, client, amount, and income source. Auto-detect repeat clients. Mark orders as cancelled with reasons. Recalculate cancellation rate by all and each income source.
- Manage Expenses: View, edit, and delete expenses with date, type, and amount.
- Daily Summary: Add, view, edit, and delete daily summaries to track progress.
- Business Notes: Add notes with dates. These will display on the income graph.
- Competitor Analysis: Add, view, and manage competitor information with their pricing and review data.
- AI Insights: AI-powered feature to tool offer insights and recommendations based on business data, incorporating trends in income sources.

## Style Guidelines:

- Primary color: Deep sky blue (#4DB6AC) for professionalism and trust.
- Background color: Light gray (#F0F4F3) to create a clean and calm atmosphere.
- Accent color: Teal (#009688) to emphasize interactive elements and important metrics.
- Font pairing: 'Space Grotesk' (sans-serif) for headers and 'Inter' (sans-serif) for body text to provide a clean and modern look.
- Use minimalist, line-based icons to represent different business metrics and data points.
- Use a grid-based layout to maintain consistency and alignment. Employ card-based UI elements to display summarized information for easy scanning.
- Subtle transitions and animations on data updates and navigation to provide a smooth user experience.
